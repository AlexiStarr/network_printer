# 打印机控制系统 - 功能总结

## ✨ 系统概览

这是一个**企业级网络打印机控制系统**，支持完整的打印任务管理、硬件监控、用户认证和实时通知。

```
┌─────────────────────────────────────┐
│         Web前端 (HTML/JS)            │ 新建: printer_control_improved.html
├─────────────────────────────────────┤
│   Go后端 (localhost:8080)            │ API服务器、认证、队列管理
├─────────────────────────────────────┤
│   C驱动 (localhost:9999)             │ 硬件模拟、状态管理
└─────────────────────────────────────┘
```

## 🔌 API端点速查表

```bash
# 认证
POST   /api/login                  # 登录
POST   /api/logout                 # 登出

# 状态
GET    /api/status                 # 获取打印机状态
GET    /api/queue                  # 获取打印队列
GET    /api/stats                  # 获取统计信息

# 任务
POST   /api/submit                 # 提交任务
POST   /api/cancel                 # 取消任务
POST   /api/pause                  # 暂停任务
POST   /api/resume                 # 恢复任务

# 维护
POST   /api/refill-paper           # 补充纸张
POST   /api/refill-toner           # 补充碳粉
POST   /api/clear-error            # 清除错误
POST   /api/simulate-error         # 模拟错误

# 历史与查询
GET    /api/history                # 获取打印历史

# 用户管理（仅管理员）
GET    /api/users                  # 列出用户
POST   /api/users                  # 添加用户
DELETE /api/users                  # 删除用户

# WebSocket
GET    /ws                         # WebSocket连接

# 其他
GET    /health                     # 健康检查
```

## 🚀 快速启动命令

```bash
# 1. 编译驱动
cd driver && gcc -o printer_driver printer_simulator.c driver_server.c main_driver.c -lpthread

# 2. 启动驱动
./printer_driver &

# 3. 启动后端
cd ../backend && go run main.go &

# 4. 打开前端（选择一个）
open printer_control_improved.html           # macOS
xdg-open printer_control_improved.html       # Linux
start printer_control_improved.html          # Windows
```
